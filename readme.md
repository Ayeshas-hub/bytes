## 📖 Project Overview

FoodBytes is a dynamic, interactive restaurant website built using HTML, Tailwind CSS, and JavaScript. It allows users to browse the menu, add/edit/delete items, search and filter results, and toggle between dark and light mode.

---

## ✨ Key Features

- ✅ Full CRUD operations on menu items (Array of Objects)
- ✅ Dark/Light Mode with localStorage persistence
- ✅ Search + 5 filters + sorting functionality
- ✅ 10 String Methods for data formatting and cleaning
- ✅ Advanced Array Methods (map, filter, sort, reduce, find, includes, every, some)
- ✅ Object manipulation (add, delete, check properties)
- ✅ Responsive design using Tailwind CSS
- ✅ Contact form integrated with Formspree API
- ✅ Modular folder structure with separate JS files

---

## 🛠️ Technologies Used

- HTML5 (Semantic)
- Tailwind CSS (via CDN)
- JavaScript ES6 (Arrow Functions, Spread Operator, Template Literals)
- Formspree API (Contact Form)
- localStorage (Theme Persistence)
- GitHub Pages (Deployment)
